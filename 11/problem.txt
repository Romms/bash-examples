h1. Скрипт додавання нового віртуального хоста для Apache2 сервера

Написати скрипт який буде запитувати назву доменого імені та виконувати для нього:
1. Створити VirtualHost
2. Додати символьне посилання для "включення" нового VirtualHost
3. Створити для домену робочу дерикторію
4. Додати домен до /etc/hosts
5. Виконати оновлення налаштувань Apache2
